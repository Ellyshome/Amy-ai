支持 Linux、macOS、Windows 操作系统，需安装 Python 3.10+。

## 环境准备

**安装 uv 包管理器：**

```bash
# macOS / Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# Windows
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
```

## 安装步骤

```bash
# 1. 克隆项目
git clone https://github.com/zhayujie/chatgpt-on-wechat.git
cd chatgpt-on-wechat

# 2. 安装依赖
make install

# 3. 交互式配置
make config
```

## 管理命令

项目使用 Makefile 管理，执行 `make` 可进入交互式菜单。

| 命令 | 说明 |
| --- | --- |
| `make dev-local` | 本地开发启动（uv + python） |
| `make dev-deploy` | 开发环境部署（Docker） |
| `make start` | 启动服务 |
| `make stop` | 停止服务 |
| `make restart` | 重启服务 |
| `make status` | 查看运行状态 |
| `make logs` | 查看日志 |
| `make config` | 交互式配置 |
| `make update` | 更新代码并重启 |
| `make` | 交互式管理菜单 |

运行后默认启动 Web 控制台，访问 `http://localhost:9899/chat` 开始对话和管理 Agent。

## Docker 部署

```bash
make dev-deploy
# Web 控制台端口: 9899
```
