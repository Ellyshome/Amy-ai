---
title: 更新升级
description: CowAgent 的升级方式说明
---

## Make 升级（推荐）

在项目根目录执行即可一键升级：

```bash
make update
```

该命令会自动完成以下流程：

1. 拉取最新代码
2. 如果是 Docker 环境，重建并重启容器

## 手动升级

在项目根目录下执行：

```bash
git pull
uv sync
```

更新完成后重启服务：

```bash
make restart
```

## Docker 升级

在项目根目录下执行：

```bash
make update
```

或手动执行：

```bash
sudo docker compose -f docker/docker-compose.dev.yml pull
sudo docker compose -f docker/docker-compose.dev.yml up -d
```

<Tip>
  升级前建议备份 `config.json` 配置文件。Docker 环境下如需保留数据，可通过 volume 挂载持久化工作空间目录。
</Tip>
